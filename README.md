# st-preset-manager

[English](README_EN.md)

一个为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) / [TauriTavern](https://github.com/Darkatse/TauriTavern)「酒馆」设计的 Chat Completion 预设管理工具。基于 Vue 3 + Pinia + TypeScript 构建，打包为单文件脚本，直接注入到酒馆页面中运行。点击右下角的悬浮窗即可打开。

它提供了一个占据全屏的独立面板，让你能够**可视化地编排、编辑、搜索、预览**预设中的每一个提示词块，以及绑定在预设上的正则脚本，而不再需要在酒馆原生的小弹窗里逐个翻找。

---

## 基本功能

- **提示词块编辑**：预设的增删改查、排序、启用/禁用、分组、搜索替换、变量追踪、跨预设复制，以及基于酒馆真实渲染管线的精确预览。
- **正则脚本编辑**：编辑绑定在预设上的正则表达式脚本（查找/替换/修剪/作用范围/深度），带本地模拟预览。
- **中 / 英文界面**：设置里可随时切换。

标签页多开、多选（Ctrl/Shift/长按）、拖拽排序、语法高亮等是两类数据通用的基础交互，细节见下方「功能详解」，不逐条展开。

---

## 安装与使用

有两种方式，分别是`使用酒馆助手(Tavern Helper)导入脚本`和`酒馆(SillyTavern)原生扩展程序`。

### 使用酒馆助手(Tavern Helper)导入脚本

1. 安装酒馆助手(详见[酒馆助手介绍](https://n0vi028.github.io/JS-Slash-Runner-Doc/guide/%E5%85%B3%E4%BA%8E%E9%85%92%E9%A6%86%E5%8A%A9%E6%89%8B/%E4%BB%8B%E7%BB%8D.html))
2. 新建一个脚本
3. 在`脚本内容`中填写`import 'https://cdn.jsdelivr.net/gh/FTWsGit/ST_PresetManager@latest/dist/index.iife.js'`，这样会从GitHub上面拉取项目构建产物，也就是这个工具。
4. 启动脚本，右下角会出现一个悬浮窗，点击即可进入。

### 使用酒馆(SillyTavern)原生扩展程序

0. 本质上需要你把这个仓库放到`SillyTavern/data/default-user/extensions/`文件夹下面
1. 一般来说，可以在SillyTavern中，打开`扩展程序`，点击`安装扩展程序`，填入这个仓库的URL(`https://github.com/FTWsGit/ST_PresetManager`)，然后安装即可。
2. 右下角会出现一个悬浮窗，点击即可进入。

## 功能详解

### 提示词块编辑

预设读写通过 SillyTavern 的 `PresetManager` API，支持操作**任意**预设，不限于当前选中的那一份。

- **分组**：多个块可绑定为可折叠分组，组内子块保持相对顺序；保存时扁平化为酒馆兼容格式，不影响其它工具正常读取。
- **多选**：桌面端 Ctrl+Click / Shift+Click，移动端长按（约 200ms）；选中状态和视觉高亮完全统一，不会出现"选了但没高亮"或反过来的情况。
- **搜索与替换**：跨所有块全文搜索，支持逐条跳转和替换。
- **拖拽排序**：自定义鼠标事件实现，不依赖浏览器原生拖拽——原生拖拽在 Tauri/WebView2 宿主下会被系统级拖拽直接截走，所以完全没用浏览器自带的那一套。

### 正则脚本编辑

编辑预设自带的正则脚本（`extensions.regex_scripts`，随预设一起保存，不需要单独存取）：查找/替换内容、修剪字符串、作用范围、生效深度等参数均可视化配置，内置查找/替换/修剪的本地模拟预览，方便调整时即时看效果。

**局限性**：预览是纯本地模拟，不解析宏（`substituteRegex`）、不体现作用范围与深度限制——这些依赖真实的酒馆上下文，本地模拟做不到完全一致，编辑器里会有文字提示这一点，预览结果仅供参考。

### 精确预览

两种预览模式均基于酒馆自身渲染管线的**真实结果**，不是自己模拟宏替换：

- **按块精确预览**：dry-run 生成，展示每个块经宏/变量/正则处理后的真实渲染文本，新增/替换的内容会高亮标出。
- **完整请求原文**：真实触发一次生成，在请求发给 API 前拦截完整 `messages` 数组并立即中断，用于最终确认插件、连接档案等对请求体的二次加工结果。

（目前只对提示词块生效，正则脚本编辑暂时没有接入这两种预览。）

### 变量追踪

自动扫描所有块中的 `{{setvar::name::value}}` / `{{addvar::name::value}}` / `{{getvar::name}}` 用法：

- **变量导航面板**：右侧列出所有变量操作，按变量名分组，支持过滤和逐条跳转。
- **点击悬浮窗**：点击编辑器中任意变量名，在光标附近弹出悬浮窗，列出该变量在当前预设中的全部出现位置。

### 跨预设复制

独立的复制工具面板，左右两栏可分别加载不同的预设，选中块（支持多选）后一键复制到对面。

---

## 已知限制

- 正则脚本目前没有多选/批量操作/分组，也还没接入精确预览和全文搜索，这些能力目前只有提示词块编辑有。
